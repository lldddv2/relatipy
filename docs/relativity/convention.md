# Convenciones de Relatividad General y Notación
Este documento establece las convenciones y notaciones utilizadas en el proyecto **KerrPy** para asegurar consistencia y claridad en los cálculos y representaciones.

## 1.1. Convenciones de Signo
- **Firma del Métrico**: Se utiliza la firma $(+,-,-,-)$ para el métrico de espacio-tiempo. ([¿De qué se trata?](https://en.wikipedia.org/wiki/Metric_signature))

## 1.2. Convención de indices
Durante todo el proyecto, se emplean las siguientes convenciones para los índices:
- **Índices de Espacio-Tiempo**: Letras griegas (μ, ν, ρ, σ) que varían de 0 a 3, donde 0 representa la coordenada temporal y 1, 2, 3 las espaciales.
- **Índices Espaciales**: Letras latinas (i, j, k, l) que varían de 1 a 3, representando las coordenadas espaciales únicamente.
